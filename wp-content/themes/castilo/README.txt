== Description ==
Fully responsive unique theme that looks great on any device.

A 'get started' documentation can be found:
 - online via https://docs.liviucerchez.com/castilo/
 - in the full package .zip file from ThemeForest

== Installation ==
1. In your admin panel, go to Appearance -> Themes and click the 'Add New' button.
2. Click on the 'Upload Theme' button following the instructions to upload the .zip file.
3. Click on the 'Activate' button to use your new theme right away.
4. Navigate to Appearance > Customize in your admin panel and customize to taste.
